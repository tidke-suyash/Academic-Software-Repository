# Academic Software Repository 

A centralized, browser-based digital platform that provides students with direct access to essential academic software required for coursework and practical learning. Stop wasting lab time and avoid risky, unofficial downloads.

## The Workflow

The system runs on a simple, straightforward concept:
1. **Curate:** Faculty gathers official, verified download links for required course software.
2. **Centralize:** The links and software details are embedded into a single, highly-organized digital repository.
3. **Access:** Students simply open the directory and click to be taken directly to the safe, official download page.

## What It Does

* **One-Click Redirection:** Directs students instantly to official software download pages, completely eliminating the risk of downloading malware or wrong versions from third-party sites.
* **Organized Library:** Software is structured and categorized so students can easily find the exact tools needed for their specific coursework.
* **Smart Search:** Quickly filter the repository by app name, description, or type using the built-in search bar.
* **Zero Hosting Required:** Runs entirely as a lightweight local HTML file. No backend, database, or server infrastructure is needed.
* **Standardized Environment:** Ensures entire classes are using the exact same, correct version of the software, drastically saving lab setup and troubleshooting time.

## How to Run It

Literally just open the file.

1. Download or clone `Repository.html`.
2. Double-click to open it in any modern web browser (Chrome, Edge, Safari, etc.).
3. Browse or search for the required software.
4. Click **INITIALIZE** on any app card to instantly route to the official download link!
